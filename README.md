
# NChat

Chat Application with Full Javascript as main language


## Features

- User Register and Login
- User Send/Receive Friend Request
- Personal Chat
- Group Chat
- Sharing Status
- Push Notification
- And many more (coming soon)


## Tech Stack

**Server:** Node, Express, MongoDB
